# Como rodar?

1.  Abra o terminal:
``` 
mkdir projects
```
2.  Clone o projeto:
```
cd projects
git clone https://github.com/mateussays/squidit.git
```
  3.  Instale as dependências:
```
cd squidit
npm i
```
  4.  Por último abra o arquivo index.html no seu navegador
